# {meta.title}

The Storybook source is used to pull individual stories from Storybook, based on tags.

The Mosaic source will filter your Storybook's stories, based on `kind` or `tags`.

For each matching story, a page is created in the Mosaic file-system.

- additional Mosaic tags can be added to each page using `additionalTags`
- additional Mosaic metadata can be added to each page using `additionalData`

This information can be used to create a dynamic index of stories from Storybook.

## Installation

`yarn add @jpmorganchase/mosaic-source-storybook`

## Configuration

The options passed to the Source

| Property       | Optional | Description                                  | Required |
| -------------- | -------- | -------------------------------------------- | -------- |
| schedule       | Y        | schedule of refresh in millisecs             | No       |
| prefixDir      | N        | prefix of pages created by source            | Yes      |
| proxyEndpoint  | Y        | proxy URL                                    | No       |
| requestTimeout | Y        | request timeout in millisecs (default: 5000) | No       |
| stories        | N        | array of story configs                       | No       |

`stories` is an array of Storybooks that you want to pull stories from

| Property       | Description                              | Required |
| -------------- | ---------------------------------------- | -------- |
| url            | url of the storybook                     | Yes      |
| description    | description of the storybook stories     | Yes      |
| filter         | RegExp filter to match on                | No       |
| filterTags     | Array of Storybook tags to match on      | No       |
| additionalTags | additional tags to add to the stories    | No       |
| additionalData | additional data to add to matching pages | No       |

### Example Local Folder Source Definition

```
sources: [
    {
      disabled: process.env.NODE_ENV !== 'development',
      modulePath: '@jpmorganchase/mosaic-source-storybook',
      namespace: 'salt',
      schedule: { checkIntervalMins: 2 },
      options: {
        prefixDir: '/salt/internal/community-index/story-navigation',
        stories: [
          {
            url: 'https://storybook.saltdesignsystem.com',
            description: 'Navigation patterns created in the Salt Labs',
            additionalData: {
                someAdditionalData: 'some-value' // can be any additional metadata you want added to the page's meta.data
            },
            additionalTags: ['salt-pattern-navigation'],
            filter: /Lab\/Tabs/ // this is a Regexp that matches on Storybook kind
          }
        ]
      }
    }
]
```

This source will pull stories from the Salt Design system storybook and add additional tags and data to each story, whose
path matches the `filter` Regexp.

Alternatively you can match on `filterTags` and the source will match on Storybook tags instead.
