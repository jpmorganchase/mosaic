# Mosaic

## Build mosaic

```
yarn
yarn build
```

## Example

`yarn example-nextjs`

Open your browser, navigate to http://localhost:8080/example/index

## CLI

### Generate Static Doc

`yarn mosaic build --out ./build --config <path/to/pd.config>`

### Serve

`yarn mosaic serve --port 8080 --config <path/to/pd.config>`
